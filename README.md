# Coffee House - Premium Landing Page ☕

Welcome to **Coffee House**, a modern, fully interactive, and premium landing page designed for coffee enthusiasts. This entire project is built dynamically using **Vanilla JavaScript (DOM Manipulation)**, showcasing how powerful and fluid a web page can be without relying on heavy frontend frameworks.

---

## ✨ Features

* **🎯 Smart Smooth Navigation:** Top navbar links (`Home`, `Menu`, `About`, `Contact`) dynamically calculate coordinates and scroll smoothly to the respective page sections.
* **🎪 Interactive Dropdown Menu:** Hoverable multi-level navigation dropdown for fast access to popular coffee categories.
* **⏳ Animated Hero Slider:** A beautifully timed automated carousel that transitions seamlessly through hero images, catchphrases, and descriptions every 4 seconds.
* **🛍️ Dynamic Order Modal:** Click "Order Now" to open a sleek, blur-background popup form equipped with active form handling and confirmations.
* **📋 Detailed Menu Modals:** Each menu card includes a "More Details" action that instantly fetches and renders comprehensive specifications and dynamic high-res product photos inside an overlay container.
* **⭐ Testimonials & Core Values:** Organized sections showcasing verified customer reviews, ratings, and unique store features (like in-house roasting and organic sourcing).
* **🎨 Warm & Premium Aesthetics:** Uses the elegant *Poppins* typeface blended into a rich espresso, latte, and cream color palette (`#2B1810`, `#6F4E37`, `#F5EBE0`, `#FFFDFA`).

---

## 🛠️ Built With

* **HTML5** (Structural foundation)
* **CSS3** (Fluid grid layouts, transitions, custom animations, and glassmorphism filters)
* **JavaScript (ES6+)** - *100% Dynamic DOM Creation & Style Injection*
